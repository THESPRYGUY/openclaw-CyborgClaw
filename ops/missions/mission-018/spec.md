# Mission 018 - Golden Run Surface Convergence Follow-On

Goal:
Use the follow-on Golden Run to converge neighboring operator surfaces only
after Mission 017 makes the `/golden-run` board SSOT-ready.

## Mission objective

Mission 018 is Part 2 of the two-part sequence.

It should only be admitted once Mission 017 proves:

- active mission anchoring is truthful
- provenance / freshness are honest
- the Golden Run systems check can fail correctly

Then Mission 018 can focus on:

1. neighboring surface convergence
2. canonical telemetry grammar
3. cross-surface agreement between Golden Run and related operator boards

## Why not now

Part 2 is intentionally blocked behind Part 1 because convergence work on top of
an untrustworthy Golden Run board would compound drift rather than reduce it.

## Scope

In scope after Mission 017:

- cross-surface telemetry agreement
- Golden Run adjacent board convergence
- canonical operator grammar for related surfaces

Out of scope until Mission 017 closes:

- activation before SSOT readiness
- broad redesign unrelated to telemetry truth

## Success bar

Mission 018 should only begin once Mission 017 can truthfully act as the
Golden Run SSOT.
