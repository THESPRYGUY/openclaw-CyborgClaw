# Quick Reference Card

## Default order

1. direct
2. plan-first
3. subagent-assisted recon/review
4. cloud delegation with local review

## Core rule

One milestone per prompt.

## Always

- preserve user changes
- validate after mutation
- close out with receipts + one next action

## Use subagents for

- exploration
- risk review
- docs / contract verification

## Avoid subagents for

- tiny fixes
- overlapping local writes
- destructive or ambiguous work
