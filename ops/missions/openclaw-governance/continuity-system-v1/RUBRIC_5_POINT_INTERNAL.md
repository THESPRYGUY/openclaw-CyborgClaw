# 5-Point Internal Rubric

## Purpose

Score the mission pack after each review pass so the chassis gets tighter instead of bloating.

## Scoring rule

Score each category from 1 to 5.

- 1 = weak, confusing, risky, or missing
- 3 = usable but soft or incomplete
- 5 = clear, tight, repeatable, and audit-safe

## Categories

1. Mission Clarity
2. Execution Readiness
3. Proof and Auditability
4. Handoff Continuity
5. Safety and Scope Control

## Pass bar

- target average: 4.5 / 5 or better
- no category below 4 / 5

## Output format for each pass

- scores by category
- main weakness
- exact revisions made
- new average
