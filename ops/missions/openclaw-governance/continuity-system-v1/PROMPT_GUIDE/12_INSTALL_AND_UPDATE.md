# 12 — Install and Update

## Install

1. Create or open the mission folder.
2. Add mission-root files:
   - `AGENTS.md`
   - `MISSION_PACK.md`
   - `HANDOFF.md`
3. Copy this pack into `PROMPT_GUIDE/`.
4. Start with `07_STARTUP_AND_REANCHOR_PROMPTS.md`.

## Update cadence

Update this pack when:

- Codex official workflow guidance materially changes
- model recommendations change
- subagent behavior or cloud workflow materially changes
- repeated mission evidence shows a file is too bulky, too thin, or duplicative

## Optional enhancements

- add OpenAI docs lookup support if your environment uses MCP / docs tooling
- add `.codex/agents/` only when you are ready to standardize optional role files

## Maintenance rule

Keep the pack lean.
If a file begins duplicating another file, merge or trim it.
